Firebase Sync Setup
Room-based sync — free ✓
Host creates a match → gets a 6-character room code → share with viewers → they join and see live updates.

5-minute setup

1. Go to console.firebase.google.com
2. Add project → name it → Continue
3. Disable Google Analytics → Create
4. Build → Firestore Database → Create → Test mode
5. Project Settings (gear icon) → General → scroll to "Your apps"
6. Add Web App → copy apiKey and projectId
7. Paste both below → Save
8. Also: Authentication → Sign-in → Anonymous → Enable
